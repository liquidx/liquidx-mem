WANDERLOOP/product family
DESIGN SYSTEM
v2.0 · terminal / hudtype · spline sans monoaccent · #A78BFAplatforms · mobile + desktop web
A sharp, minimal terminal aesthetic for the Wanderloop family of location apps — Check-In, Atlas, Timeline and future products. Monospace at small sizes, near-black surfaces, hairline structure, and a single purple accent used sparingly. Tokens, type, iconography and components are shared; each product composes them into its own screens across mobile and desktop web.
§00PRINCIPLES
01 / sharp
No rounded corners. Structure is drawn with 1px hairlines, ticks and boxed cells.
02 / airy
Lots of black space. Few elements per view. Density comes from data, not chrome.
03 / sparing accent
Purple marks one thing per view: the peak, the live point, or the primary action.
04 / labels vs values
UPPERCASE tracked labels; lowercase values. The mono grid does the rest.

Everything animates once on view load and is replayable. Motion is sharp and mechanical — no bounce, no easing overshoot beyond the draw curve. It should read like a terminal booting, not a consumer app.
§07DESKTOP WEBsidebar · columns · breakpoints
The same tokens, type and components carry to the browser. The change is spatial, not stylistic: the bottom tab bar becomes a left sidebar, the single mobile column becomes a fixed nav plus a fluid content region, and horizontal space is spent on parallel panels rather than stacked scroll. No new colors, no rounded corners, no shadows introduced for desktop.

§08PRODUCT FAMILYshared core · per-app surface
Wanderloop is a family, not one app. Every product draws from the same core — tokens, Spline Sans Mono, the single purple accent, hairline structure, Tabler nav icons and the component set. What differs per app is which screens exist and how the core is composed. The accent hue is the one lever a product may retune to distinguish itself; everything else is fixed.
Check-In
#A78BFA · purple
the flagship — map, timeline, places, check-in. Reference implementation of the system.
Atlas
#6BB0A0 · teal
shared trips & group location. Desktop-first planning board, mobile companion.
Timeline
#D0A24C · amber
personal history & stats. Histogram + spine components take the lead role.
future
pick 1 accent hue
new products inherit the core verbatim and choose a single accent at the same chroma/lightness.
